# Code Security Analyzer

AI-powered security vulnerability detection for JavaScript and TypeScript.

## Features

- Real-time vulnerability detection (hardcoded secrets, SQL injection, XSS)
- AI-powered explanations using OpenAI
- Security dashboard with vulnerability statistics
- Supports JavaScript and TypeScript files

## Requirements

- OpenAI API key for AI-powered explanations

## Configuration

Configure your OpenAI API key in VS Code settings:

- Open Settings (Cmd+,)
- Search for "Code Security Analyzer"
- Enter your OpenAI API key
